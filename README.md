# 🌍 Экологическая платформа Казахстана + Система авторизации

Веб-ресурс, разработанный для освещения экологических проблем Казахстана (Аральское море, Каспий, смог в мегаполисах) и координации эко-волонтеров. Проект сочетает в себе адаптивную верстку и серверную часть для управления профилями пользователей.

---

## ✨ Функционал приложения

* **Информационные страницы:** Интеграция текстовых и графических материалов по экологии регионов на чистом HTML/CSS.
* **Регистрация пользователей:** Сбор и валидация данных (ФИО, Email, телефон) и их запись в базу данных.
* **Аутентификация (Вход):** Серверная проверка пароля и авторизация волонтера.
* **Автоматический Fallback аватара:** Если пользователь не загрузил фото, PHP на уровне сессии подставляет стандартную иконку-заглушку, предотвращая поломку верстки.

---

## 🛠 Технологический стек проекта

* **HTML5** — семантическая структура страниц, создание форм ввода данных (Вход / Регистрация).
* **CSS3** — стилизация интерфейса, оформление блоков, кнопок, таблиц и адаптивный дизайн под экраны.
* **PHP 8** — обработка входящих HTTP-запросов методом `POST`, работа с сессиями и архитектурная логика бэкенда.
* **SQLite 3** — встроенная реляционная база данных. Вся база хранится локально в одном файле `database.db`.

---

## 🚀 Инструкция по развертыванию

1.  Скопируйте файлы проекта в корневую папку локального сервера (например, **OpenServer** в папку `domains/`).
2.  Проверьте файл конфигурации `php.ini` — должно быть включено расширение `pdo_sqlite`.
3.  Запустите локальный сервер и откройте проект в браузере. Файл базы данных `database.db` и таблица `users` создадутся автоматически при первой отправке формы.
